# Use Cases

## AI Research Bot

A bot watches public sources and drafts research notes. Agent Evidence Board forces each note into a structured card with claims, sources, risks, and unknowns before publication.

## OSS Maintainer Automation

A maintainer uses agents to summarize issues, release notes, or security advisories. The gate blocks summaries that lack source links, contain secret-like values, or overstate weak evidence.

## Security Alert Drafting

A security researcher drafts public alerts from public evidence. The gate blocks stale social posts, preview-only sources, and unsupported claims before an alert is published.

## Crypto/Market Research Notes

Agents can collect public market or project signals, but cards must separate confirmed facts, weak signals, assumptions, risks, and unknowns. The gate blocks guarantee language and unsupported claims.

## What This Enables

- More auditable agent outputs.
- Better handoff between multiple agents and humans.
- Safer publication workflows.
- Reproducible examples for reviewing agent failures.
